# **DealUp** – Java Projekts

**DealUp** ir programma, kurā cilvēki var izlikt savas pārdodamās lietas, un citi cilvēki var tās apskatīt un iegādāties. Projekta galvenais mērķis ir nodrošināt vienkāršu un ērtu platformu preču tirdzniecībai.

## **Funkcijas:**

1. **Personas "Sign In" opcija**
   - Lietotāji var izveidot kontu un pieteikties sistēmā, lai piekļūtu personalizētām funkcijām.

2. **Meklēšanas opcija**
   - Lietotāji var meklēt preces pēc nosaukuma, kategorijas, cenas utt.

3. **Filtrēšanas opcija**
   - Lietotāji var filtrēt preces pēc dažādiem parametriem, piemēram, cena, stāvoklis, atrašanās vieta utt.

4. **Uzrakstīt e-pastu**
   - Lietotāji var sazināties ar pārdevējiem, nosūtot e-pastu tieši no programmas.

5. **Apskatīt talruņa nr. un e-pastu**
   - Lietotāji var redzēt pārdevēja kontaktinformāciju, lai sazinātos ar viņu tieši (ja pārdevējs to ir piekritis publicēt).

6. **Apskatīt savus sludinājumus**
   - Lietotāji var apskatīt un pārvaldīt visus sludinājumus, kurus viņi ir ievietojuši platformā.

## **Darbības Norise:**

Projekts var tikt papildināts ar papildu funkcijām, piemēram, maksājumu sistēmu, reģistrāciju ar sociālajiem tīkliem, lietotāju atsauksmēm, utt.
